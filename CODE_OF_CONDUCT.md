# Code of Conduct

## Expected Behavior

Contributors are expected to be respectful, patient, and constructive. Assume good intent, welcome questions, give actionable feedback, and keep discussion focused on improving Quizz.

## Unacceptable Behavior

Unacceptable behavior includes harassment, threats, discriminatory language, sustained disruption, personal attacks, publishing private information without consent, and any conduct that makes participation unsafe or unwelcome.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, reviews, documentation, and maintainer communication related to Quizz.

## Reporting Path

Report conduct concerns to the project maintainers through an appropriate private maintainer channel when one is available. For private or security-sensitive matters, follow [SECURITY.md](SECURITY.md). Do not include sensitive personal or vulnerability details in a public issue.

## Enforcement Responsibilities

Maintainers are responsible for clarifying expectations, reviewing reports, taking proportionate action, and protecting reporters from retaliation where possible.

## Enforcement Process

Maintainers may ask participants to edit or remove content, pause a conversation, issue a warning, limit participation, or block participation in project spaces. Enforcement decisions should be based on the severity and pattern of behavior, available evidence, and the need to keep the project safe and productive.
